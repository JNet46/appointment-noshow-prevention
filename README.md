# Healthcare Appointment No-Show Prevention System

## Overview
ML-powered web app that predicts appointment no-shows

## Features
- CSV upload
- Risk prediction (0.0-1.0)
- Interactive analytics
- Downloadable reports

## Installation
bash
pip install -r requirements.txt
streamlit run app.py

## Usage
1. Upload appointment CSV
2. Click "Generate Predictions"
3. View risk scores and analytics
4. Download results
